After completing Treehouse's Front End Web Development Track, including 54 hours of HTML, CSS, and JavaScript courses, I decided to put my new skills to the test and make a new version of my portfolio website from scratch. I started by creating a project plan with three main phases:

  • Phase 1: Create a minimum viable product that is a direct copy of my previous weebly site
  • Phase 2: Get and implement feedback from Veterans United's UX/UI Director
  • Phase 3: Use Figma to create a redesigned prototype, then bring that to life

I am currently at the beginning of Phase 3. Check out the files on GitHub.
